# FocusFlow AI

FocusFlow AI is an AI-powered productivity assistant that helps users manage tasks, habits, and schedules using Google Gemini.

## Features

- AI-generated daily plans
- Task management
- Habit tracking
- Smart rescheduling
- AI-powered recommendations

## Tech Stack

- HTML
- CSS
- JavaScript
- Google Gemini API
