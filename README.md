# 🚀 FocusFlow – AI-Powered Productivity Coach

FocusFlow is an AI-powered productivity web application that helps students and professionals organize their work intelligently. Instead of acting as a traditional to-do list, FocusFlow analyzes tasks and habits to generate personalized productivity plans using Google's Gemini AI.

## ✨ Features

### 📊 Dashboard

* Personalized productivity dashboard
* AI-generated daily productivity plans
* AI insights and recommendations
* Progress tracking
* Active tasks overview
* Habit summary

### ✅ Task Management

* Add new tasks
* Edit existing tasks *(in progress)*
* Delete tasks
* Priority management
* Due dates
* Estimated completion time

### 🔥 Habit Tracking

* Habit monitoring
* Streak tracking
* Productivity overview

### 🤖 AI Coach

* Gemini AI integration
* Personalized daily plans
* Smart productivity recommendations
* Intelligent task prioritization

## 🚀 Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install dependencies

```bash
npm install
```

3. Create a `.env` file

```env
GEMINI_API_KEY=YOUR_API_KEY
```

4. Start the server

```bash
node server.js
```

5. Open the frontend using Live Server or a local web server.
